semantiva.tools.export_ontology module

Export the Semantiva component taxonomy and metadata to RDF/Turtle using live framework metadata. Sample usage:

python semantiva/tools/export_ontology.py -o semantiva_ontology.ttl -p semantiva,semantiva_imaging
