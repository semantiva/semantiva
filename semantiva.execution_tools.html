class semantiva.execution_tools.tasks.PayloadOperationTask(payload_source_class, payload_source_parameters, payload_operation_class, payload_operation_config, payload_sink_class=None, payload_sink_parameters=None)

Bases: ComputingTask

Task for processing payloads using a data source, a payload operation, and a data sink.

This class encapsulates the logic for: - Retrieving data and context from a data source. - Applying a payload operation to the data and context. - Sending the processed data and context to a data sink.
